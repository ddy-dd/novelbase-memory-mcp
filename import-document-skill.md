# import-document — 文档导入技能

将 Word（.docx）或 PDF 文档的内容转化为结构化 Markdown 文件，供 novelbase pipeline 导入知识图谱。

## 支持的格式

以下章节可以在每个 `.md` 文件中使用：

| 章节 | 作用 | 生成节点类型 |
|------|------|------------|
| `## 角色` / `## 人物` | 提取角色及其属性 | Character 节点 + AppearsIn 边 |
| `## 地点` / `## 场所` | 提取故事场景 | Location 节点 |
| `## 关系` / `## 关系图谱` | 创建角色之间的边 | Knows 边（含描述） |
| `## 情节脉络` / `## 结构` | 标记开端/发展/高潮/结局 | Scene 节点 + PartOf 边 |
| `## 冲突` / `## 矛盾` | 记录主要矛盾冲突 | Note 节点 + Mentions 边 |
| `## 环境` / `## 背景` | 设定时代/社会/风俗/文化 | Note 节点 |

## 角色格式（详细版）

```markdown
## 角色
- 本名：别名1、别名2（身份，性格，外貌：描述，动机：描述，心理：描述，结局：描述）
```

各字段说明：

| 字段 | 说明 | 例子 |
|------|------|------|
| 本名 | 角色正式姓名 | `林黛玉` |
| 别名 | 逗号分隔的别号 | `颦儿、潇湘妃子` |
| 身份 | 角色社会身份 | `小姐` |
| 性格 | 性格特征 | `多愁善感，才华横溢` |
| 外貌： | 外貌描写 | `闲静时如姣花照水` |
| 动机： | 角色核心动机 | `追求知己之爱` |
| 心理： | 心理活动描述 | `敏感多疑` |
| 结局： | 角色最终结局 | `泪尽而逝` |

简洁版（向后兼容）：

```markdown
## 角色
- 张三：剑客，性格勇敢
- 李四：书生
```

## 地点格式

```markdown
## 地点
- 地点名：描述
- 大观园：贾府为省亲所建，四季皆景
- 潇湘馆：林黛玉住所，多竹，幽静
```

## 关系格式

```markdown
## 关系
- 角色A → 角色B：关系描述
- 贾宝玉 → 林黛玉：表兄妹，木石前盟
- 贾宝玉 → 贾母：祖孙
```

## 情节脉络格式

```markdown
## 情节脉络
- 阶段名：回目范围，描述
- 开端：1-5回，人物出场
- 发展：6-80回，矛盾积累
- 高潮：81-100回，家族衰败
- 结局：101-120回，树倒猢狲散
```

## 冲突格式

```markdown
## 冲突
- 甲方 vs 乙方 → 矛盾本质
- 贾宝玉 vs 贾政 → 仕途理想与个人自由的冲突
```

## 环境格式

```markdown
## 环境
- 时代：清朝乾隆年间
- 社会：封建大家族，等级森严
- 风俗：元宵节、中秋节、诗社
- 文化：诗词、戏曲、园林
```

## 完整示例

```markdown
# 第一回 甄士隐梦幻识通灵

## 角色
- 贾宝玉：怡红公子、宝二爷（公子，叛逆多情，外貌：面若中秋之月，动机：追求真情，心理：厌恶仕途经济，结局：出家为僧）
- 林黛玉：颦儿、潇湘妃子（小姐，多愁善感，外貌：闲静时如姣花照水，动机：追求知己之爱，心理：敏感多疑，结局：泪尽而逝）

## 地点
- 大观园：贾府省亲所建
- 潇湘馆：林黛玉住所

## 关系
- 贾宝玉 → 林黛玉：表兄妹，木石前盟

## 环境
- 时代：清朝乾隆年间
- 社会：封建大家族

## 情节脉络
- 开端：1-5回，人物出场

## 冲突
- 贾宝玉 vs 贾政 → 仕途与自由的矛盾
```

## 导入命令

```bash
# 1. 将文档转为本章格式的 .md 文件（手动）
# 2. 导入项目
cargo run -- cli import novelbase/ --project 红楼梦

# 3. 搜索验证
cargo run -- cli search 贾宝玉 --project 红楼梦

# 4. 启动 Web UI
cargo run -- ui --port 8080
```
